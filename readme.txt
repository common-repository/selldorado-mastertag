== Description ==

Selldorado’s module installs automatically on your e-commerce shop the technical elements required to launch campaigns on Selldorado.
So, you can directly launch your advertising campaigns on Selldorado without technical intervention.

What Selldorado does for tou : A few notes about the sections above:


*    Save time : Selldorado proposes package deals for your needs and can be activated immediately
*    A qualified traffic : with few clicks, access to a powerful and recognised partner network that broadcast your ads (iGraal , ma-reduc.com , Google, Ebuyclub ... )
*    Simplicity : a single tool to manage your online advertising campaigns
*    Fast : quick activation and intuitive to use, even for novices
*    No subscription required and no fixed fees, you only pay commission on sales provided by the partners
*    Advertising medium hosting included
*    Access to help documentation and our community to develop your knowledge in the e-commerce field


With this simple module to configure, you can install automatically and quickly:

*    The Selldorado sales tracer
*    The Selldorado partners container

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/selldorado-mastertag` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Selldorado Mastertag screen to configure the plugin
1. Fill-in your advertiser id
